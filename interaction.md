# Premier Design Furniture Website - Interaction Design

## User Experience Flow

### Primary User Journey
1. **Landing Page Discovery**: User arrives at homepage and sees impressive hero image of custom furniture
2. **Company Introduction**: User scrolls to learn about Premier Design's values and services
3. **Product Exploration**: User clicks "СМОТРЕТЬ КАТАЛОГ" to view furniture catalog
4. **Product Details**: User browses furniture cards with detailed descriptions and images
5. **Contact Engagement**: User clicks "ОБСУДИТЬ ПРОЕКТ" to connect via WhatsApp

## Interactive Components

### 1. Hero Section CTA Button
- **Location**: Main banner on index.html
- **Function**: "СМОТРЕТЬ КАТАЛОГ" button that navigates to catalog.html
- **Interaction**: Smooth hover effect with color transition
- **User Value**: Direct path to product exploration

### 2. Product Filter System
- **Location**: Catalog page sidebar
- **Function**: Filter furniture by category (sofas, beds, chairs)
- **Interaction**: Click category buttons to show/hide relevant products
- **User Value**: Streamlined product discovery based on user needs

### 3. Product Card Hover Effects
- **Location**: All product cards in catalog
- **Function**: Hover reveals additional product details and "ЗАКАЗАТЬ" button
- **Interaction**: 3D tilt effect with shadow expansion and overlay information
- **User Value**: Engaging product preview without leaving the page

### 4. WhatsApp Contact Integration
- **Location**: Floating contact button and footer
- **Function**: Direct WhatsApp chat integration for instant communication
- **Interaction**: Click opens WhatsApp with pre-filled message
- **User Value**: Immediate connection for custom orders and consultations

## Navigation Structure
- **Header Navigation**: Home | Catalog | About | Contacts
- **Mobile Menu**: Hamburger menu for responsive design
- **Footer Links**: Social media, contact info, quick links

## Responsive Design Considerations
- Mobile-first approach for furniture browsing
- Touch-friendly buttons and cards
- Optimized image loading for various screen sizes
- Smooth scrolling and transitions for better UX

## Accessibility Features
- Alt text for all furniture images
- Keyboard navigation support
- High contrast text on images
- Clear call-to-action buttons
